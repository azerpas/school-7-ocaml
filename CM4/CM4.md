## Tris
```py
[(11,  "Arthur", "France" ) ; (8,  "Lilia", "Suisse" ) ; (20,  "Mani", "Espagne" ) ; (2,  "Eva", "France" ) ]
```

L'intérêt d'un tri est de pouvoir accéder à 
l'information de façon plus efficace, en 
utilisant le critère du tri.

### Spécification
- trier : Liste(E) -> Liste(E, <)
                l  -> l' contenant
